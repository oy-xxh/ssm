# tmall_springboot
springboot+vue.js 仿天猫整站
项目描述： 
  本项目使用了springboot，vue.js，shiro，Redis，elasticsearch等一整套技术栈，实现了前后端分离。
  前端：首页，商品页，结算页面，购买页面，支付页面，登录注册页面等
  后端：分类管理，分类属性管理，商品管理商品属性值管理，商品图片管理，用户管理，订单管理
